# Doing to play play

This project is a clicker game created with ReactJS and inspired by [Torsumathp](https://github.com/Torsumathp/Doing_to_play_play).

> [!IMPORTANT]
> - **Install nodeJS**
> ```
> npm install
> ```
> - **Run Project**
> ```
> npm run dev
> ```


> [!WARNING]
> ### Responsive Project
> 
> **✅ PC / Laptop** <br>
> **❌ Mobile**
